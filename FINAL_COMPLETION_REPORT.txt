Q: Docker not running?
A: Start Docker Desktop (Windows/Mac) or:
   sudo systemctl start docker (Linux)

Q: Port already in use?
A: See DOCKER_PRODUCTION.md "Port already in use" section

Q: Container won't start?
A: Check logs: docker-manage.ps1 logs backend

Q: MongoDB connection error?
A: Restart: docker-manage.ps1 restart mongodb

Q: Need to change passwords?
A: Edit .env.docker and restart:
   docker-manage.ps1 restart backend
